# Cat Kingdom

Repository for my Cat Kingdom Game.

# Getting Started

* This is strategy game about gathering resources, making buildings and facing off enemy waves.

* The main menu shows the highest scores among users and allows you to choose between 2 levels.

* Order your units to gather resources by right clicking on them. Once the "Remaining Time" is over, all workers switch to fighters.

# Running

In the executable folder you can find a runnable version of the application.
